Display the beautiful WhatsApp Sticky Button on the WordPress frontend.

== Description ==

WP Sticky Button is a Free & Light Weight WordPress plugin that displays the beautiful WhatsApp Sticky Button on the WordPress frontend.
After clicking on the WhatsApp Sticky Button, It will bring you to the WhatsApp application or WhatsApp website.

1. It will bring you to the WhatsApp application if WhatsApp application installed on your device
2. It will bring you to the WhatsApp website  if WhatsApp application is not installed on your device

**Admin Setting Panel**

* WhatsApp Sticky Button - Settings
1. Activate/Deactivate WhatsApp Sticky Button
2. Customize “Display on” functionality for the Mobile, Tablet & Desktop Devices
3. Change the Position of WhatsApp Sticky Button to Top-Left/Top-Right/Bottom-Left/Bottom-Right
4. Add WhatsApp Number
5. Add WhatsApp Default Message
6. Customize Icon Width
7. Customize Icon Height
8. Customize Icon Margin
9. Change Icon Type to Default WhatsApp Icon/Custom Icon Image
10. Select Custom Icon
11. Save & Enjoy!!!

== Installation ==

1. Just log into your WordPress admin dashboard
2. Visit Dashboard > Sidebar > Plugins > Add New
3. Search for “WP Sticky Button”
4. Install and Activate the “WP Sticky Button” plugin
5. Visit Dashboard > Sidebar > WhatsApp
6. Activate the WhatsApp Sticky Button to display on frontend (Mandatory)
7. Add a valid WhatsApp number & default message (Mandatory)
8. Set “Display on” functionality for the Mobile, Tablet & Desktop devices
9. Customize the Position, Width, Height, Margin, Type and Image of Icon (Optional)
10. Use & Enjoy!!!
